# 책일기 : 소셜 독서 기록 서비스

- 목표 독서량 설정
- 책 검색하여 추가
- 책에 코멘트 작성, 비밀 코멘트 설정
- 친구 검색
- 친구 책장 놀러가기
- 추가한 책 기반 책 추천
